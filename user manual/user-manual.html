WELCOME!

Genesis-21 welcomes you!
Genesis-21 is a desktop application that helps you retrieve the origin and the entire instances of the post in any Social Media platform.
This manual is designed to give you an overview of the various features of Genesis-21.

Dashboard

The dashboard displays the details of the previous searches.

  • number of posts analysed
  • number of posts reported

Activity

Activity holds the particulars about the on-going search. It acknowledges us on the number of agents assigned and the details about the batch process (depending upon the system�s configuration such as RAM capacity, network speed and stability,etc. ), number of posts retrieved and the time taken for it by the respective agent.

Manage search

Manage search gives us a lead on the status of all the searches ever made by the user.
The status bar is classified as follows:

  • Successfully completed searches are tagged as INFO. On clicking info, you will be able to see the report of the search.
  • On-going searches are tagged as IN-PROGRESS.
  • Searches that are to be made further are tagged as PENDING
  • Searches that are yet to be initialized are tagged as INITIALIZE.

Configuration

Our software is user-friendly. Our software displays the different configurations of the user�s device and the user has the choice to change accordingly to enhance efficiency of the search.
Configuration displays the details on the Operating system, system�s memory and storage capacity and the speed of the internet connection.

New search

How this works:

The spotlight of our user interface is the NEW SEARCH feature. This clickable button once instigated, guides the user to a page where the user is asked to select the desired platform among the following:
INSTAGRAM
TWITTER
REDDIT.

After choosing the desired platform, it follows to another page where the user is asked to enter the URL of the post or upload the available content.

The entered data is stored in our firebase database and the execution process begins.

Report

All the matched data is retrieved atlast and a report is generated that contains a brief description of the origin and other branches.

PDF Genertion

Additionally, the user is given an option to generate a PDF. The PDF contains all the posts from the origin to all the branches that ever posted the content.
